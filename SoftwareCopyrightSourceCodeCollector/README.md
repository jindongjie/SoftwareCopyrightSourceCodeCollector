## 快速开始

### 前提条件

- .NET 8 SDK
- 一个 IDE，如 Visual Studio


### 使用 NativeAOT 发布

要使用 NativeAOT 发布项目，运行：

```dotnet publish -c Release -r win-x64 --self-contained```

将 `win-x64` 替换为 `osx-x64` 或 `linux-x64` 以适应其他平台。
